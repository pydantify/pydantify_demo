# pydantify_demo
Simple pydantify demo

```
pip install pydantify rich ruff
git clone https://github.com/YangModels/yang.git

pyang -f tree yang/standard/ietf/RFC/ietf-interfaces.yang yang/standard/ietf/RFC/ietf-ip.yang

pydantify yang/standard/ietf/RFC/ietf-interfaces.yang yang/standard/ietf/RFC/ietf-ip.yang


```
